# Inventory Management Portal

This project is an Inventory Management Website designed to streamline and manage inventory operations efficiently. Below are the key features and details of the portal:

## Project Overview
- **Purpose:** Manage inventory, user registration, and assignment of inventory items to users.
- **Technology Stack:**
  - Backend: PHP (with database management via phpMyAdmin)
  - Database: MySQL
  - Frontend: Web-based portal

## Features
- User registration and authentication
- Assign inventory items to users
- View and manage assigned inventory
- Upload inventory updates via CSV files
- Convert ODS files to CSV for easy upload
- Database management using phpMyAdmin

## CSV Upload
- The portal allows uploading CSV files to update inventory records in the database.
- Uploaded CSV files are processed and the inventory is updated accordingly.

## Getting Started
1. Clone this repository.
2. Set up your PHP and MySQL environment (phpMyAdmin recommended).
3. Convert your ODS inventory file to CSV (you can use LibreOffice or online tools).
4. Upload the CSV file via the portal to update the inventory.
